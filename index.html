Liquid is fluidifying SoLiD

We believe that the current steep learning curve for both users and developers can be a barrier to widespread adoption of SoLiD. That's why we're dedicated to enrich SoLiD ecosystem with more approachable tools.

Projects

  1. SOLID PROXY

    Bridging Solid and traditional web by allowing users without Pods to consume Solid apps

  2. LIQUID UI

    A UI for the Community Solid Server based on a user-facing version of Penny

  3. LIQUID POD

    A Pod provider based on the Community Solid Server and using LIQUID_UI

  4. SOLID APP STORE

    A repository of Solid apps

  5. FedCM

    FedCM can improve Solid UX, We advocate for a more decentralized version of it, and experiment with Solid-OIDC compatibility
